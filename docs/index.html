Python

Python é uma linguagem de programação versátil conhecida por sua simplicidade e legibilidade.

"Programa simples:" print("Olá, Mundo!")

Este programa imprime a mensagem "Olá, Mundo!" no console.

JavaScript

JavaScript é uma linguagem de programação amplamente utilizada para desenvolvimento web.

// Programa simples: // console.log("Olá, Mundo!");

Este programa registra a mensagem "Olá, Mundo!" no console.

Java

Java é uma linguagem de programação popular utilizada para construir uma ampla variedade de aplicativos.

// Programa simples: // public class HelloWorld { public static void main(String[] args) { System.out.println("Olá, Mundo!"); } }

Este programa imprime a mensagem "Olá, Mundo!" no console.

C#

C é uma linguagem de programação amplamente utilizada, conhecida por sua eficiência e desempenho.

/* Programa simples: */ #include int main() { printf("Olá, Mundo!\n"); return 0; }

Este programa imprime a mensagem "Olá, Mundo!" no console.

C++

C++ é uma poderosa linguagem de programação amplamente utilizada para desenvolver aplicativos complexos.

// Programa simples: // #include int main() { std::cout << "Olá, Mundo!" << std::endl; return 0; }

Este programa imprime a mensagem "Olá, Mundo!" no console.
